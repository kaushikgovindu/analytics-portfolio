# Analytics Portfolio

Supply Chain Analytics Portfolio Website - Retail KPI Reporting & Operational Intelligence

## 📊 Overview

This repository contains a professional portfolio website showcasing supply chain analytics capabilities built with **Power BI**, **SQL**, and **Python**. The portfolio demonstrates KPI reporting, data modeling, and visualization skills for a national office supplies retailer.

**Live Portfolio:** https://kaushikgovindu.github.io/analytics-portfolio

## 📁 Files

- **index.html** - Main portfolio website with responsive design
- **exec-summary.png** - Executive Summary dashboard screenshot
- **fc-performance.png** - Fulfillment Center Performance dashboard screenshot  
- **replenishment.png** - Replenishment & Inventory dashboard screenshot

## 🎯 Project Highlights

### Three Interactive Dashboards:

1. **Executive Summary** - High-level KPI snapshot
   - Stockout Rate: 181.96K aggregate %
   - On-Time Delivery: 754.73K %
   - Transport Cost per Unit: 231.97K
   - Regional trend analysis (Eastern vs Western)

2. **Fulfillment Center Performance** - Operational drill-down
   - Fill Rate % by FC
   - Perfect Order % by FC
   - Regional performance disparities

3. **Replenishment & Inventory** - Tactical planning view
   - Stockout hotspot identification
   - Inventory turnover analysis
   - Days of supply metrics

## 📊 Dashboard Insights

- **Stockout Risk:** 181.96K cumulative rate indicates inventory optimization opportunities, especially in Eastern region
- **Strong Logistics:** 754.73K on-time delivery reflects solid execution with regional variance for optimization
- **Cost Pressure:** Transportation at 231.97K per unit is a key focus area; high-volume lanes need analysis
- **Regional Trend:** Fill rate trending upward 2024→2026 in both regions, with Eastern region showing steeper recovery
